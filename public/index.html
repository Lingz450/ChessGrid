ChessGrid

Massive Chess Game for Farcaster

Getting Started

This is a Farcaster Frame chess game. To use it:

  1. Deploy this server to a public URL (see README.md)
  2. Share the URL on Farcaster/Warpcast
  3. Play chess directly in your feed!

Features

  • Full chess game with move validation
  • Multiplayer support
  • Solo play mode
  • Move history tracking
  • Check/Checkmate detection
  • Beautiful visual board

How to Play

Making Moves: Enter moves in standard notation:

  • e2e4 - Pawn from e2 to e4
  • g1f3 - Knight from g1 to f3
  • e7e8q - Pawn promotes to queen
